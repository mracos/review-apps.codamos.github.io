Sobre os eventos

O Codamos está comprometido a dar destaque a eventos que prezem pela inclusão, diversidade e acessibilidade. Criamos rótulos para facilitar a busca por eventos que têm ícones representados pelas legendas abaixo:

  • = incentivo para mulheres
  • = tem código de conduta
  • = incentivo pessoas trans
  • = acesso para cadeirantes
  • = tem tradução para libras
  • = apoio para deficientes visuais
  • = on-line, para todos os públicos
  • = tem espaço/atividades infantis
  • = incentivo comunidade negra
  • = incentivo para comunidade LGBT
  • = tem acessibilidade
  • = incentivo comunidade periférica
  • = evento indicado para iniciantes
  • = inclusivo para restrições alimentares
